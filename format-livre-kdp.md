# Format KDP retenu pour les premiers livres

## Choix de production

- Plateforme : Amazon KDP
- Type : livre broche / paperback
- Format interieur : 15,24 x 15,24 cm
- Equivalent pouces : 6" x 6"
- Nombre de pages : 24 pages
- Interieur : couleur
- Couverture : souple, imprimee en couleur
- Objectif : format compact, joli pour un livre enfant et plus accessible a produire qu'un grand carre 20,96 x 20,96 cm

## Pourquoi ce format

Ce format permet de commencer avec un livre enfant simple a tester :

- assez petit pour rester abordable ;
- carre, donc adapte aux couvertures et illustrations deja preparees ;
- facile a presenter sur TikTok, Instagram et la page web ;
- compatible avec une collection de plusieurs titres ;
- suffisant pour une histoire courte ou un livre illustre de premier test.

## Point a verifier avant publication

Avant l'envoi final sur KDP, confirmer dans l'interface Amazon :

1. que le format personnalise 15,24 x 15,24 cm est bien accepte pour le paperback ;
2. que l'option couleur choisie correspond au rendu souhaite ;
3. que le cout d'impression et le prix minimum de vente restent compatibles avec l'objectif "pas cher" ;
4. que le PDF interieur et la couverture respectent les marges et fonds perdus KDP.
